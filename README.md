# Automated Daily Work Report System

## Overview

Automated backend system built using Supabase, PostgreSQL, Edge Functions, Gmail SMTP, and Cron Jobs to send daily work report emails automatically.


# Features

- Daily work tracking
- Automated email reports
- Gmail SMTP integration
- Cron job scheduling
- PostgreSQL functions
- Supabase Edge Functions


# Tech Stack

- Supabase
- PostgreSQL
- TypeScript
- Gmail SMTP
- pg_cron
- GitHub


# Workflow

Employee Tasks
      ↓
PostgreSQL
      ↓
Edge Function
      ↓
Gmail SMTP
      ↓
Automated Email
